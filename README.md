## README
### Description

Website that helps longboarders find and share media and locations to ride.

### Setup
1. Clone the project
2. _bundle install_
3. _rake db:setup_
4. _rails s_

More thorough instructions for vagrant can be found on the [wiki pages](https://github.com/andyklimczak/bomb-hills/wiki/Vagrant).
